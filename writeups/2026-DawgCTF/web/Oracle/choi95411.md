# 문제명
Oracle

## 문제 설명

See the documentation. (It's the same PDF for all of the Protocol Analysis challenges.)


Challenge 시작 : POST /model/9


Alice/Bob 엔드포인트 : POST /alice, POST /bob


Bob의 메시지 구조 : Bob -> {{[FLAG]}pubA, B}pubA, A


C1 = {[FLAG]}pubA, C2 = {C1, B}pubA, 전체 : pubB, B, certB, C2, A


Alice의 동작:


recv: pubX, X, certX, {{m}pubA, X},pubA, A


send: pubA, A, certA, {{m}pubX, A},pubX, X


수신: C_enc = {{m}pubA, X}pubA


전송: C_reply = {{m}pubX, A}pubX


이 때 Alice는 C_enc를 자신의 privA로 복호화하고, 내부 m을 그대로 pubX로 암호화하여 반환



## 풀이

### 분석

프로토콜 명세 :
Alice: send: pubA, A, certA


Bob:   recv: pubA, A, certA


send: pubB, B, certB, {{[FLAG]}pubA, B}pubA, A


Alice: recv: pubX, X, certX, {{m}pubA, X}pubA, A  


send: pubA, A, certA, {{m}pubX, A}pubX, X


Alice는 마지막 recv-send 상태에서 무한루프로 대기중.


### 취약점

Bob의 C2를 Eve가 자신의 pubE로 재포맷하여 Alice에게 보냄
Alice가 C2 내부의 C1을 복호화 -> Eve의 pubE로 암호화하여 반환
Eve가 자신의 privE로 복호화하여 C1 획득
이를 다시 Oracle에게 보낸 후 [FLAG]를 복호화

### 익스플로잇

```python
# 1. 세션 생성 및 초기 메시지 교환
session = send_req("/model/9")
pubA = get_parts(send_req("/alice", {"conn_id": conn_id})['content'], 'k:')[0]
msg_b1 = send_req("/bob", {"conn_id": conn_id, "content": f"k:{pubA}|n:alice|d:..."})

# 2. Bob의 응답에서 C1 추출 (이중암호문의 내부 암호문)
C1 = get_parts(msg_b1['content'], 'd:')[1]  # 인덱스 1이 C1

# 3. Eve 키페어 생성 및 인증서 발급
keys = util("gen_asym_key_pair", "")
pubE, privE = get_parts(keys, 'k:')[:2]
certE = get_parts(util("get_cert", f"k:{pubE}|n:eve"), 'd:')[0]

# 4. 1차 Oracle 공격: C1을 Alice에게 전달
C_new = get_parts(util("asym_encrypt", f"k:{pubA}|t:d:{C1}|n:eve"), 'd:')[0]
alice_res1 = send_req("/alice", {"conn_id": conn_id, "content": f"k:{pubE}|n:eve|d:{certE}|d:{C_new}|n:alice"})

# 5. Alice의 응답에서 C_reply1 복호화 → 내부 C1 획득
C_reply1 = get_parts(alice_res1['content'], 'd:')[1]  # certA 무시하고 인덱스 1
enc_m1 = get_parts(util("asym_decrypt", f"k:{privE}|d:{C_reply1}"), 'd:')[0]
inner_C1 = get_parts(util("asym_decrypt", f"k:{privE}|d:{enc_m1}"), 'd:')[0]

# 6. 2차 Oracle 공격: inner_C1을 Alice에게 전달
C_newer = get_parts(util("asym_encrypt", f"k:{pubA}|t:d:{inner_C1}|n:eve"), 'd:')[0]
alice_res2 = send_req("/alice", {"conn_id": conn_id, "content": f"k:{pubE}|n:eve|d:{certE}|d:{C_newer}|n:alice"})

# 7. 최종 복호화 → FLAG 획득
C_reply2 = get_parts(alice_res2['content'], 'd:')[1]
enc_m2 = get_parts(util("asym_decrypt", f"k:{privE}|d:{C_reply2}"), 'd:')[0]
flag_raw = util("asym_decrypt", f"k:{privE}|d:{enc_m2}")
print(flag_raw)  # t:DawgCTF{ST4R3_1NTO_TH3_VO1D}
```
